# Sweetleaf

Welcome to the Sweetleaf project! This is a frontend development project created for my frontend development class. Sweetleaf is a salad bar restaurant website designed to showcase my skills in HTML, CSS, and JavaScript.

## Live Website

Check out the live website here: [Sweetleaf](https://shaderia-s.github.io/Sweetleaf-2-/)

## Project Description

Sweetleaf is a mock website for a salad bar restaurant. The goal of this project was to create a responsive and user-friendly website that provides information about the restaurant, its menu, and its services.

### Features

- **Responsive Design:** The website is designed to be fully responsive and looks great on all devices.
- **Interactive Menu:** Users can browse the menu and see detailed descriptions of each item.
- **Contact Form:** A functional contact form that allows users to send inquiries.

### Technologies Used

- **HTML5:** For the basic structure and content of the website.
- **CSS3:** For styling and layout.
- **JavaScript:** For interactive elements and form validation.
